What you read is what executes

mach has no hidden runtime and no hidden allocation. The entry point is ordinary code: you name the symbol and pull in the runtime yourself. Nothing runs that you did not write.

use        std.runtime;
use print: std.print;

#[symbol("main")]
fun main(argc: i64, argv: **u8) i64 {
    print.println("Hello, World!");
    ret 0;
}
